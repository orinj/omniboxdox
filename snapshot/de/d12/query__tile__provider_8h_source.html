24 // An autocomplete provider that asynchronously provides matching
25 // query tiles to build the query tiles suggestion row according to the
26 // following.
27 // (1) For zero suggest, top level query tiles will be displayed.
28 // (2) If user taps on a query tile, the suggestion will be replaced by the next
29 // level query tiles for the selected tile.
30 // (3) If user starts typing, the query tile suggestion will be dismissed.
