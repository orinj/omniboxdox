36 // This caches favicons by both page URL and icon URL. We cache a small number
37 // of them so we can synchronously deliver them to the UI to prevent flicker as
38 // the user types.
39 //
40 // This class also observes the HistoryService, and invalidates cached favicons
41 // and null responses when matching favicons are updated.

50  // These methods fetch favicons by the |page_url| or |icon_url| respectively.
51  // If the correct favicon is already cached, these methods return the image
52  // synchronously.
53  //
54  // If the correct favicon is not cached, we return an empty gfx::Image and
55  // forward the request to FaviconService. |on_favicon_fetched| is stored in a
56  // pending callback list, and subsequent identical requests are added to the
57  // same pending list without issuing duplicate calls to FaviconService.
58  //
59  // If FaviconService responds with a non-empty image, we fulfill all the
60  // matching |on_favicon_fetched| callbacks in the pending list, and cache the
61  // result so that future matching requests can be fulfilled synchronously.
62  //
63  // If FaviconService responds with an empty image (because the correct favicon
64  // isn't in our database), we simply erase all the pending callbacks, and also
65  // cache the result.
66  //
67  // Therefore, |on_favicon_fetched| may or may not be called asynchronously
68  // later, but will never be called with an empty result. It will also never
69  // be called synchronously.
70  //
71  // Note that GetFaviconForPageUrl and GetLargestFaviconForPageUrl should not
72  // be used interchangeably. These methods use the same |page_url| key for
73  // caching favicons and as a result may return favicons with the wrong size if
74  // called with the same |page_url|.
75  gfx::Image GetFaviconForPageUrl(const GURL& page_url,
76  FaviconFetchedCallback on_favicon_fetched);
77  gfx::Image GetLargestFaviconForPageUrl(
78  const GURL& page_url,
79  FaviconFetchedCallback on_favicon_fetched);
80  gfx::Image GetFaviconForIconUrl(const GURL& icon_url,
81  FaviconFetchedCallback on_favicon_fetched);
