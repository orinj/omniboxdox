42  // Given some string |text| that the user wants to use for navigation,
43  // determines how it should be interpreted.
44  // |prefer_keyword| should be true the when keyword UI is onscreen; see
45  // comments on AutocompleteController::Start().

52  // |match| should be a non-NULL outparam that will be set to the default
53  // match for this input, if any (for invalid input, there will be no default
54  // match, and |match| will be left unchanged). |alternate_nav_url| is a
55  // possibly-NULL outparam that, if non-NULL, will be set to the navigational

57  // AutocompleteResult::alternate_nav_url_ in autocomplete.h.
