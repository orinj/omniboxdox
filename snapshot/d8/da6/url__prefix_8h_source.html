32  // Sees if |text| is inlineable against either |input| or |fixed_up_input|,
33  // returning the appropriate inline autocomplete offset or
34  // std::u16string::npos if |text| is not inlineable.
35  // |allow_www_prefix_without_scheme| says whether to consider an input such

37  // needed because sometimes the string we're matching against here can come

40  // inlineable. ("www." is not otherwise on the default prefix list.)
