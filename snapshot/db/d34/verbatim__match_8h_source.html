17 // Returns a verbatim match for input.text() with a relevance of
18 // |verbatim_relevance|. If |verbatim_relevance| is negative then a default
19 // value is used. If the desired |destination_url| is already known and a
20 // |history_url_provider| is also provided, use |destination_description| as the
21 // description. Providing |history_url_provider| also may be more efficient (see
22 // implementation for details) than the default code path.
23 // input.text() must not be empty.

32 // Returns a match representing a navigation to |destination_url|, highlighted
33 // appropriately against |input|. |trim_default_scheme| controls whether the
34 // match's |fill_into_edit| and |contents| should have the scheme (http or
35 // https only) stripped off, and should not be set to true if the user's
36 // original input contains the scheme. The default scheme is https if |input|
37 // is upgraded to https, otherwise it's http.
38 // NOTES: This does not set the relevance of the returned match, as different
39 // callers want different behavior. Callers must set this manually.
40 // This function should only be called on the UI thread.
