21 // Splits |search| into words and finds them in |text|, returning a vector of
22 // occurrence starts and ends.
23 // - Occurrences must be sequential. E.g. 'a c' can be found in 'a b c' but not
24 // in 'c b a'.
25 // - Occurrences must be at word breaks. E.g. 'a c' cannot be found in 'a bc'.
26 // - Whitespaces must also match. E.g. 'a c' cannot be found in 'a-c' but can
27 // be found in 'a -c' and 'a- c'.
28 // If all words in |search| were not found, then returns an empty vector.
29 std::vector<std::pair<size_t, size_t>> FindWordsSequentiallyAtWordbreak(
30  const std::u16string& text,
31  const std::u16string& search);

33 // Inverts and reverses term |matches| in a domain of [0, |length|) to determine
34 // the selected non-matches. Ranges are interpreted as {start, end}. E.g., if
35 // |length| is 10 and |matches| are {{2, 3} {5, 9}}, |InvertRanges| will return
36 // {{10, 9}, {5, 3}, {2, 0}}. Assumes |matches| is in forward order; i.e.
37 // |matches[i+1]| occurs after |matches[i]| and |matches[i].second| after
38 // |matches[i].first|.
39 std::vector<gfx::Range> TermMatchesToSelections(
40  size_t length,
41  std::vector<std::pair<size_t, size_t>> matches);
