19 // An asynchronous autocomplete provider which receives input string and tries
20 // to find the matches in an on device head model. This provider is designed to
21 // help users get suggestions when they are in poor network.
22 // By default, all matches provided by this provider will have a relevance no
23 // greater than 99, such that its matches will not show before any other
24 // providers; However the relevance can be changed to any arbitrary value by
25 // Finch when the input is not classified as a URL.
26 // TODO(crbug.com/925072): make some cleanups after removing |model_| and |this|
27 // in task postings from this class.
